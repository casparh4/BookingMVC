# Booking MVC 
### My ASP.Net MVC application 

booking MVC is a hotel booking site similar to booking.com etc

## features

- AI tools built using Semantic Kernel and OpenAI
  - to return relevant hotels based on query searches
  - make readable summaries from hotel reviews
  - ChatBot for answering customer questions relevant to bookingMVC
    
- real and relevant flight options from FlightStackAPI
  
- Admin-only access pages to add and remove hotels
   - made with ASP.NET Core Identity
 
- SqlServer Database
   - stored Hotels, Reviews, users and bookings

- Moq Tests
